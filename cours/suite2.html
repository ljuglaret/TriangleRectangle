Théorème

Dans un triangle ABC rectangle en B : AB²+BC²=AC²

Demonstration

Soit F le cercle de centre O et de rayon r = OA = OC = OB . D'après le théorème précedent, le triangle ABC est rectangle en B. On écrit les distances AB, BC, AC chacune élevée au carré AC est un diamètre du cercle F,donc $$\begin{align*} & AC^{2} \ =\ ( 2*r)^{2} \ =\ 4*r^{2} \ \\ &AB^{2} =( x_{B} \ -x_{A})^{2} \ +\ ( y_{B} -y_{A})^{2}\\ &BC^{2} =( x_{C} \ -\ x_{B} \ )^{2} \ +\ ( y_{C} -y_{B})^{2}\\ \end{align*}$$

le point C est l'opposé du point A par la symétrie de centre O $ \left\{ \begin{array}{l} x_C=-x_A\\ y_C=-y_A \end{array} \right\} $

Donc $$\begin{align*} &BC^{2} =( -x_{A} \ -\ x_{B} \ )^{2} \ +\ ( -y_{A} -y_{B})^{2}\\ &\Longrightarrow \ BC^{2} =( -1*( x_{A} \ +\ x_{B})) \ ^{2} \ +\ \ ( -1\ *\ ( y_{A} \ +\ y_{B}))^{2}\\ &\Longrightarrow \ BC^{2} =( -1)^{2} \ *\ ( x_{A} \ +\ x_{B})^{2} \ +\ ( -1)^{2} \ *\ ( y_{A} \ +\ y_{B})^{2}\\ &\Longrightarrow \ BC^{2} =\ ( x_{A} \ +\ x_{B})^{2} \ +\ ( y_{A} \ +\ y_{B})^{2}\\ &\end{align*}$$

Donc $$\begin{align*} &AB^{2} \ +\ BC^{2} =( x_{B} \ -x_{A})^{2} \ +\ ( y_{B} -y_{A})^{2} +\ ( x_{A} \ +\ x_{B})^{2} \ +\ ( y_{A} \ +\ y_{B})^{2}\\ &\Longrightarrow \ AB^{2} \ +\ BC^{2} =x_{B}^{2} \ -2*x_{A} *x_{B} +\ x_{A}^{2} \ +y_{B}^{2} \ -2*y_{A} *y_{B} +\ y_{A}^{2}\\ &\Longrightarrow \ AB^{2} \ +\ BC^{2} =\ x^{2}_{A} \ +\ 2*x_{A} *x_{B} +\ x_{B}^{2} \ +y_{A}^{2} \ +2*y_{A} *y_{B} +\ y_{B}^{2}\\ &\Longrightarrow AB^{2} +\ BC^{2} =\ x_{B}^{2} +x_{A}^{2} \ +y_{B}^{2} +\ y_{A}^{2} +\ x^{2}_{A} \ +\ x_{B}^{2} \ +y_{A}^{2} +\ y_{B}^{2}\\ &\Longrightarrow AB^{2} +\ BC^{2} =\ 2*\left( x_{B}^{2} +x_{A}^{2}\right) \ +\ 2*\left( y_{B}^{2} +\ y_{A}^{2}\right)\\ &\Longrightarrow AB^{2} +\ BC^{2} =\ 2*\left(( x_{B} \ -x_{O})^{2} +( x_{A} \ -x_{O})^{2}\right) \ +\ 2*\left(( y_{B} \ -y_{O})^{2} +( y_{A} \ -y_{O})^{2}\right)\\ &\Longrightarrow AB^{2} +\ BC^{2} =\ 2*\left(( x_{B} \ -x_{O})^{2} \ +( y_{B} \ -y_{O})^{2}\right) +2*\left(( x_{A} \ -x_{O})^{2} \ +\ ( y_{B} \ -y_{O})^{2}\right) &\end{align*}$$
