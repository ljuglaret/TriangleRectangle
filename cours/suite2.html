Théorème de Pythagore

Dans un triangle ABC rectangle en B : AB²+BC²=AC²

Première méthode

Soit F le cercle de centre O et de rayon r = OA = OC = OB . D'après le théorème précedent, le triangle ABC est rectangle en B. On écrit les distances AB, BC, AC chacune élevée au carré AC est un diamètre du cercle F,donc $$\begin{align*} & AC^{2} \ =\ ( 2*r)^{2} \ =\ 4*r^{2} \ \\ &AB^{2} =( x_{B} \ -x_{A})^{2} \ +\ ( y_{B} -y_{A})^{2}\\ &BC^{2} =( x_{C} \ -\ x_{B} \ )^{2} \ +\ ( y_{C} -y_{B})^{2}\\ \end{align*}$$

Donc
$\begin{align*} AB^{2}+BC^{2} &= (x_B-x_A)^2 + (y_B-y_A)^2 + (x_C-x_B)^2 + (y_C-y_B)^2\\ \Longrightarrow \ AB^{2} \ +\ BC^{2} &=(x_B-x_O+x_O-x_A)^2 + (y_B-x_O+x_O-y_A)^2 + (x_C-x_O+x_O-x_B)^2 + (y_C-y_O+y_O-y_B)^2\\ \Longrightarrow \ AB^{2} \ +\ BC^{2} &=(x_B-x_O)^2+(x_O-x_A)^2 + (y_B-y_O)^2+(y_O-y_A)^2 \\ &+ (x_C-x_O)^2+(x_O-x_B)^2 + (y_C-y_O)^2+(y_O-y_B)^2\\ &+ 2*(x_B-x_O)*(x_O-x_A)+2*(y_B-y_O)*(y_O-y_A)+ 2*(x_C-x_O)*(x_O-x_B)+2*(y_C-y_O)*(y_O-y_B)\\ \Longrightarrow \ AB^{2} \ +\ BC^{2} &=(x_B-x_O)^2+(y_B-y_O)^2+(x_O-x_A)^2 +(y_O-y_A)^2 \\ &+ (x_C-x_O)^2+ (y_C-y_O)^2+(x_O-x_B)^2+(y_O-y_B)^2\\ &+ 2*(x_B-x_O)*((x_O-x_A)+*(x_O-x_C)) + 2*(y_B-y_O)*(y_O-y_A+y_O-y_C)\\ \Longrightarrow \ AB^{2} \ +\ BC^{2} &=r^2+r^2 +r^2 + r^2\\ &+ 2*(x_B-x_O)*(2*x_O-x_A-x_C) + 2*(y_B-y_O)*(2*y_O-y_A-y_C)\\ \end{align*}$

Le point C est l'opposé du point A par la symétrie de centre O $ \left\{ \begin{array}{l} \frac{x_A+x_C}{2}=x_O\\ \frac{y_A+y_C}{2}=y_O\\ \end{array} \right\} \Longrightarrow \left\{ \begin{array}{l} 2*x_O=x_C+x_A\\ 2*y_O=y_C+y_A\\ \end{array} \right\} $

$\begin{align*} \Longrightarrow \ AB^{2} \ +\ BC^{2} &=r^2+r^2 +r^2 + r^2\\ &+ 2*(x_B-x_O)*0 + 2*(y_B-y_O)*0\\ \Longrightarrow \ AB^{2} \ +\ BC^{2} &=4*r^2+r^2 +r^2 + r^2=BC^2 &\end{align*}$

Deuxième méthode

L'aire du carré ABCD est : $\begin{align*} (a+b)^2 &\end{align*}$

L'aire des triangles AEH, BEF, CFG, DGH, rescpectivement rectangles en A, B, C, D, est : $\begin{align*} frac{a*b} {2} &\end{align*}$

Donc l'aire du carré EFGH est : $\begin{align*} (a+b)^2 - 4*\frac{a*b} {2} = a^2+2*a*b+b^2-2*a*b=a^2+b^2 &\end{align*}$

Donc $\begin{align*} c = \sqrt{a^2+b^2} &\end{align*}$

Donc $\begin{align*} c^2 = a^2+b^2 &\end{align*}$

Donc dans un triangle rectangle le carré de l'hypoténuse est égale à la somme des carrés des deux autres côtés.
