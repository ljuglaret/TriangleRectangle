Théorème Réciproque

Si l'un des côtés d'un triangle est un diamètre de son cercle circonscrit, alors ce triangle est rectangle.
Le diamètre du cercle circonscrit est alors son hypoténuse.

Démonstration

$\begin{align*} &La\ somme\ des\ angles\ dans\ un\ triangle\ est\ égale\ à\ 180º\ donc\ :\\ \\ &180º =\widehat{BDC} \ +\ \widehat{DCB} \ +\widehat{CBD} \ \\ \\ &A,\ B\ et\ D\ sont\ des\ points\ du\ cercle\ \mathcal{A} \ de\ centre\ O\ donc\ OD=OA\ =\ OB,\\ &donc\ les\ triangles\ ODC\ et\ OCB\ sont\ tous\ les\ deux\ isocèles\ en\ O.\\ &Donc\ \widehat{ODC} \ \ =\ \widehat{DCO\ } \ =\alpha ,\ et\ \ \widehat{OCB} =\widehat{CBO} =\beta \ .\\ &Or\ :\ \ \ \widehat{DCB} =\widehat{DCO\ } \ +\ \widehat{OCB}\\ &Donc\ \ \widehat{DCB} =\alpha +\beta \\ \\ &De\ plus\ \widehat{BDC} \ \ =\widehat{ODC} \ =\alpha \ car\ les\ points\ D\ O\ et\ B\ sont\ alignés\ dans\ cet\ ordre.\\ &De\ même\ \widehat{CBD} \ =\ \widehat{CBO} \ =\beta \\ &Donc\ 180º =\alpha \ +\ \alpha +\ \beta \ +\ \beta \\ &\Longrightarrow 180º =2*\alpha +2*\beta \\ &\Longrightarrow 180º =2*( \alpha +\beta ) \ \\ &\Longrightarrow \frac{180º }{2} =\frac{2*( \alpha +\beta ) \ }{2}\\ \\ &\Longrightarrow 90º =\alpha +\beta \\ &\Longrightarrow 90º=\widehat{DCB}\\ &Donc\ le\ triangle\ DBC\ est\ rectangle\ en\ C \end{align*}$
